WiX to WordPress automated data migration plugin. Convert WiX to WordPress easily and fast.

== Description ==

You’re running a website on WiX. For some reason it turns out that WiX is not exactly the platform you want to stay on. You’ve heard a lot about WordPress and now planning to get there. Moving from WiX to WordPress can take you years to transfer all the data? No worries, here’s the plugin to help solve this migration issue.

= How does this plugin help: =
- Preview the migration results first with running Free Demo Migration;
- Average migration time = 15 min (Note: timing depends on how many content items you’re up to move from WiX to WordPress);
- Migrate the content you want. You choose how the migrated content looks on WordPress with the help of Data ‘Mapping’ option;
- Migration price depends on the content amount you want to migrate to WordPress.

= What will be moved from WiX to WordPress?: =
* Pages and posts, content included there and content creation date;
* Images included in the content body;
* Menus and menu items;
* Taxonomy (Tags)

= Preserved SEO juice after the migration =
* Meta Data Migration (migration of keywords, metadata/meta description);
* Automated 301 redirects from existing WiX website to the new WordPress website;

= Compatible with: WordPress 4.1.x - 5.0.x. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Wix To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
